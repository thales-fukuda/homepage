# homepage
This startup page is shown every time I open a new tab.

http://thalesfukuda.me/homepage/

## Shortcuts
The search field has some shortcuts to popular websites.

- y: YouTube
- f: Facebook
- r: Reddit
- g: GitHub

### Exemple
> y:funny videos

Will search "funny videos"on YouTube.
